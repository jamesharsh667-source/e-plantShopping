# e-plantShopping

## Paradise Nursery Shopping Application

The Paradise Nursery Shopping Application is a React-based e-commerce web application that allows users to browse and purchase a variety of plants.

## Features

- Landing page with a background image
- About Us section
- Product listing page
- Add plants to the shopping cart
- Remove items from the shopping cart
- Redux Toolkit for cart state management
- Responsive user interface

## Technologies Used

- React
- Vite
- Redux Toolkit
- React Redux
- JavaScript
- CSS
